53. EVENT BASICS (CLICK EVENTS)
///////////////////////////////
Kako napraviti event listener:
1. odabrati DOM element nad kojim želimo da se event odvije
2. dodati event --> addEventListener() funckija --> "sluša" event nad elementom
3. dodati callback funckiju koja se provodi kada se event odvije

Primjer: 
const button = document.querySelector('button')
const listElement = document.querySelector('ul > li')

listElement.addEventListener('click', () => {
    console.log('Stisnut sam!')
})

- event se može dodati na svaki element, kao i na kolekciju elemenata kroz koje možemo iterirati
- kada se event dogodi u browseru, u callback funckiji dobivamo parametar 'e', koji je tipa Event object
- Event object sadrži podosta property-a
- property 'target' vraća DOM element nad kojim se event izvršio

Primjer:
const items = document.querySelectorAll('li')

items.forEach((item) => {
    item.addEventListener('click', e => {
        e.target.style.backgroundColor = 'green'
        e.target.style.border = '3px solid black'
        e.target.style.borderRadius = '10px'
    })
})

54. CREATING AND REMOVING ELEMENTS
//////////////////////////////////
Kako ukloniti DOM element:
- funkcija za uklanjanje DOM elementa sa browsera: remove()
- remove() funkcija briše određeni DOM element; ako je funckija u sklopu HTML taga koji u sebi sadrži children elemente, svi children
  elementi se pomiču za jedno mjesto 

Primjer:
const ul = document.querySelector('ul')

ul.addEventListener('click', e => {
    e.target.remove()
})

Kako kreirati novi DOM element:
- funkcija createElement('HTML tag') stvara novi DOM element, sa svim property-ima koji imaju mogućnost izmjene; funckija se poziva na 
  document u skoro svim slučajevima

Primjer:
const button = document.querySelector('button')
button.addEventListener('click', e => {
    const newListElement = document.createElement('li')
    newListElement.style.color = 'red' ---> property novonastalog elementa
})

Kako umetnuti novi DOM element u web stranicu:
- nakon što kreiramo novi DOM element, možemo ga umetnuti u web stranicu pozivom dvije funckije:
  1. append('novokreirani DOM element ili node')
  2. prepend('novokreirani DOM element ili node')
- obje funkcije se pozivaju nad elementom kojem želimo umetnuti novokreirani DOM element, a kao parametar stavljamo novokreirani DOM element
- obje funckije primaju kao parametar DOM string element ili Node objekt
- razlika: append() stavlja novokreirani element na kraj elementa (iza zadnjeg child objekta) kojem želimo umetnuti novi DOM element, 
  a prepend() na početak elementa (ispred prvog child objekta) kojem želimo umetnuti novi DOM element
